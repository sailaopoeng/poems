```
ငါမခံစားနိုင်တော့ဘူး
သက်ပြင်းတွေ တုန်ရီရှည်လျား
```

```
ငါ့ခံတပ်ဟာ ယဲ့ယဲ့လေး တွန်းလှန်နေရ
အားတင်းထားရတယ်
အဝေးမှာ
```

```
စကားလုံးတွေက ပေါ့ပျက်လွန်းတယ်ကွာ
ငါ၊ ဆက်မရေးနိုင်တော့ဘူး
```
